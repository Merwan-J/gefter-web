## Common Patterns Across All Pages

### Navigation Structure
- **Bottom Navigation**: Consistent across all pages with 4-5 items:
  - Home
  - Transactions/Actions/History
  - Split/Receipt
  - Groups (some pages)
  - Profile
- **Active State**: Current page highlighted with accent/primary color
- **Icons**: Material Symbols Outlined icons used throughout

### Header Patterns
- **Standard Header**: Back button (left), centered title, optional action button (right)
- **Sticky Positioning**: Headers are sticky with z-index 10
- **Variations**: Some pages have no header (full-screen confirmations)

### Form Patterns
- **Input Fields**: Rounded inputs with focus states
- **Amount Inputs**: Large, centered text inputs with currency prefix
- **Search Inputs**: Full-width with search icon on left
- **Date/Time Inputs**: Grid layout for optional details

### Button Patterns
- **Primary Actions**: Full-width, rounded-full, accent/primary color, bold text
- **Secondary Actions**: Gray background, rounded-full
- **Icon Buttons**: Circular buttons with Material Symbols icons
- **Action Groups**: Side-by-side buttons for binary choices (Reject/Acknowledge)

### Card Patterns
- **Transaction Cards**: Profile image, description, amount (color-coded), timestamp
- **Detail Cards**: Rounded-xl background with padding, key-value pairs
- **Status Badges**: Rounded-full badges with color coding (green=paid, yellow=pending, red=rejected)

### Status Indicators
- **Color Coding**: 
  - Green: Positive amounts, paid status, success
  - Red: Negative amounts, rejected status, errors
  - Yellow: Pending status
  - Blue: Marked as paid, informational
- **Icons**: Check circles, error icons, hourglass icons for status

### Dark Mode Support
- **Theme Switching**: All pages support dark mode via class strategy
- **Color Adaptations**: Text colors, backgrounds, and borders adapt for dark mode
- **Consistent Patterns**: Same structure maintained across light and dark themes
